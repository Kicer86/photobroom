◆ stoppableTask()

template<typename R , typename Callback >
void stoppableTask ( const std::stop_source &  stop_source,
std::function< void(const std::stop_token &, StoppableTaskCallback< R >)>  task,
Callback  callback,
bool  sameThread = true 
)

run task and then call callback with result if stop_source was not stopped

Parameters
stop_sourceused to get stop_token passed to task. task should take use of it to stop work when needed.
taskstoppable task to be executed. task should call provided callback with result whem job is done
callbackcallback provided by user to be called when task is done
sameThreadif set to true, callback will be called from the same thread as task

stoppableTask() will wrap callback using safe_callback_ctrl to make sure it won't be called when stop_source was stopped. Therefore it is safe to destroy any objects used by callback before task is done.

Note
All objects in task need to be valid during execution.
