Person::Id Database::PeopleInformationAccessor::store ( const PersonName &  pn)
finaloverridevirtual

Store or update person.

  • pn Details about person name to be stored.
    Returns
    id of created or updated person.
    If pn has valid id then person name will be updated in database
    If id is not valid and pn holds name already existing in database
    then method will return id of that person.
    If id is not valid and name was not found in database then
    new person will be added to database and its id will be returned.
