◆ TmpOption

enum System::TmpOption

Flags to be used for temporary directory creation.

Enumerator
Generic 

generic temporary dir without any special requirements. This option cannot be combined with others.

Confidential 

directory will be used for user private data. No system wide tmp dir should be used as a base.

Persistent 

use persistent location which will be reused when called for the same util name.

BigFiles 

big files will be stored in tmp dir. RAM tmp dirs won't be used.
