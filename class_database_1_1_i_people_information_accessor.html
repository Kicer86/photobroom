virtual PersonInfo::Id Database::IPeopleInformationAccessor::store ( const PersonInfo &  pi)
pure virtual

Store or update person details.

  • pi Details about person. It needs to refer to a valid photo id.
    Also at least one of rect, person or id need to be valid

If pi has valid id and rect is invalid and person is is not valid,
then information about person is removed.
if pi has invalid id then database will be searched for exisiting
rect or person matching information in pi. If found, id will be
updated and any not stored detail (rect or person) will be updated.
