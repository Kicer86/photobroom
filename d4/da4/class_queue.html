Detailed Description

template<typename Task>
class Queue< Task >

Base class for all kind of executor queues.

Purpose of this class is to have a base queue of tasks to be executed by any executor but controled by client (can be cleaned)

template<typename Task >
enum class Queue::Mode
strong

Determines queue behavior.

Enumerator
Fifo 

Execute tasks in order of insertion. First inserted will be first to execute.

Lifo 

Execute tasks in reversed order of insertion. Last inserted will be first to execute.
